About


About this site

This is a free farming control site that helps to see what do you need to do.
It is free web hosting site so it maybe be slow sometimes and in the end of the month, it could be offline because of the traffic limit. So if you like it and could help, please donate to put it in a pay web hosting.

How to use it

When you made an login account, all your data will be saved just for you. Without it, everyone can change the values in this site because I let and test account running by default. So, please don't forget to make an account and login to don't lose all your data.

When logged in, you have to put all your game data in the "Complete Farm Sheet" tab. There is where you customize all your chars farm. You can control characters Ascension Level, Talents Levels and even choose a weapon for a character with the Weapon Ascension Level.
When you check on "I have this char", this website begins to account for everything that your character needs.

With the "Complete Farm Sheet" tab customize for you, all the quantities need will be at "Backpack" tab.
You need to enter your Genshin Impact game and type all your quantities for each item here in this tab. They are all in the order of the game backpack. So, if you don't find an item it's because you don't have it yet.
All the need quantities are under the space that you need to type. When you hover over the figure of the item, you can see with character needs it.

Future Features

All the Domains of the day are shown using your computer clock. So I have to implement the Pacific Daylight Time Zone (UTC -7) on it to be right with the server.
There is a tab with each character with each control resources and some builds to let everything be in one place. It's not implement because of the traffic limit. So if we get this site in a pay web hosting, I implement it.
